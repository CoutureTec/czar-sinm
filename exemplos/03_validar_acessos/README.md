# Teste de autenticação

Utilize esse exemplo para testar suas credenciais

Use o env.exemaple como referencia, e salve um .env com suas credenciais.

Execute:

python exemplo.py 

O retorno exibira os erros de autenticação ou o resultado da consulta das permissões do seu usuário
